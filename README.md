AI Resume Analyzer & Improver

A modern, AI-powered web application that analyzes resumes, calculates ATS scores, detects missing keywords, compares resumes with job descriptions, and provides actionable improvement suggestions — all through a clean and intuitive UI.

🚀 Features

Upload Resume (PDF) & extract text automatically

ATS Score Calculator (0–100 based on keyword matching)

Job Description Comparator with missing-keyword analysis

AI-Based Improvement Suggestions for skill gaps & content quality

Clean Dashboard UI with sections for score, strengths & weaknesses

Instant Resume Summary Rewriter

Fully Frontend-Based Version for Static Hosting



---

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

AI Logic: Built-in keyword analysis + resume parsing

Hosting: GitHub Pages



---

📂 Project Structure

index.html — Main UI

style.css — Styling

script.js — Logic for ATS scoring, extraction & suggestions

assets/ — Icons, fonts, UI elements



---

⚙️ How It Works

1. Upload your resume (PDF)


2. Paste the job description


3. The system extracts keywords & content


4. Calculates ATS score based on matches


5. Highlights missing keywords


6. Shows strengths, weaknesses, and suggestions


7. Generates an improved summary block
